A production-ready Machine Learning project for predicting student mathematics performance using Scikit-learn. The project follows a modular ML pipeline and is deployed on AWS with automated CI/CD using GitHub Actions.

## 📖 Overview

This project demonstrates an end-to-end Machine Learning workflow for predicting student mathematics performance. It includes data ingestion, data transformation, model training, prediction, and deployment. The application is built using Flask and deployed on AWS with an automated CI/CD pipeline using GitHub Actions.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Akanksha-2312/ML_Project.git
```

Move into the project directory

```bash
cd ML_Project
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python application.py
```
